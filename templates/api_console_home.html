Quick Start Guide

To get access to our API, you must carefully follow and implement our quick start guide. That way you'll be able to get easy access to all of the features of our API.

Step 1

  • Create an account if you don't have one.
  • If you already have an account, then login to your account with your password and email.

Step 2

  • Once you are good to go with your account, go to the Subscribe to API section to see subscriptions.
  • Get started by subscribing to any plan that suits your needs and requirements.
  • Once you subscribe to a plan, you'll get an API Key.
    • Warning! Keep your API Key to yourself.

Step 3

  • Once you have your API key, use this link localhost:8000/get_data/?api_key=your_api_key to access the desired data
    • Warning! Keep your API key to yourself or someone may miss use your key.
